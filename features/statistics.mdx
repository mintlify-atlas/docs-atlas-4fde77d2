The Seguridad system provides comprehensive statistical analysis tools for crime data, allowing users to generate detailed reports based on various criteria.

## Overview

The statistics module enables authorized users to query crime data across different time periods, geographic regions, and crime types. The system supports both detailed incident-level reporting and aggregated statistical summaries.

## Statistical Query Features

### Date Range Filtering

Query crime data within specific date ranges:

<ParamField path="desde" type="date" required>
  Start date for the query range (format: DD-MM-YYYY or YYYY-MM-DD)
</ParamField>

<ParamField path="hasta" type="date" required>
  End date for the query range (format: DD-MM-YYYY or YYYY-MM-DD)
</ParamField>

<Warning>
  When querying with "TODOS" (all units/dependencies), the date range cannot exceed 12 months to prevent database overload.
</Warning>

### Geographic Filters

<ParamField path="ur" type="select">
  **Unidad Regional (Regional Unit)**
  
  Select a specific regional unit or "TODOS" for all units. Options are dynamically loaded from the `ur` table.
</ParamField>

<ParamField path="dependencia" type="select">
  **Dependencia (Department/Station)**
  
  Cascading dropdown that loads departments based on the selected regional unit. Disabled until a unit is selected.
</ParamField>

### Additional Filter Criteria

<ParamField path="tipo_delito" type="select">
  **Crime Type**
  
  Filter by specific crime type from the `tipo_delito` table, or select "TODOS" for all types.
</ParamField>

<ParamField path="modus_operandi" type="select">
  **Modus Operandi**
  
  Filter by the method or pattern of crime execution. Options loaded from `modus_operandi_preventivos` table.
</ParamField>

<ParamField path="esclarecidos" type="select" default="T">
  **Case Status**
  
  - `T`: TODOS (All cases)
  - `S`: SI (Solved cases only)
  - `N`: NO (Unsolved cases only)
</ParamField>

## Statistical Reports

### Crime Statistics Report

Accessed via `consultaDelito.php`, this report provides:

- Detailed incident listings
- Crime counts by type and location
- Temporal analysis of crime patterns
- Case clearance rates

### General Statistics

The `consultaEstadistica.php` module provides:

- Monthly and annual crime summaries
- Crime type breakdowns by unit
- Comparative statistics across regions
- Trend analysis over time

### Monthly Statistical Forms

The `planillas.php` system allows manual entry of monthly statistics organized by crime category, including:

- Year and month selection
- Regional unit assignment
- Detailed crime counts by category
- Solved cases (esclarecidos) tracking

<Info>
  Statistical forms use an accordion interface to organize the extensive list of crime categories, making data entry more manageable.
</Info>

## Query Execution

<Steps>
  <Step title="Select Filters">
    Choose the regional unit, department, date range, and crime type criteria for your query.
  </Step>
  
  <Step title="Apply Additional Criteria">
    Optionally filter by modus operandi and case status (solved/unsolved).
  </Step>
  
  <Step title="Execute Query">
    Click the "BUSCAR" (Search) button to generate the report. Results open in a new window.
  </Step>
  
  <Step title="Review Results">
    Analyze the detailed results table showing incidents matching your criteria.
  </Step>
  
  <Step title="Export Data">
    Use export functions to save or print the report for further use.
  </Step>
</Steps>

## Database Schema

The statistics module queries the following key tables:

- `delitos` - Main crime incidents table
- `tipo_delito` - Crime type classifications
- `ur` - Regional units
- `dependencia` - Police departments/stations
- `modus_operandi_preventivos` - Crime methods/patterns

## Performance Considerations

<Warning>
  Large queries (e.g., all units over extended periods) may take significant time to process. The system enforces a 12-month maximum for province-wide queries to maintain performance.
</Warning>
